# 目标

生成一个 prompt 管理的桌面应用, 用于创建、编辑和组织各种 AI 模型的提示词 (prompts).
所有代码保存在 src 目录下.

# 功能需求

1. **prompt 管理界面**:
   - 直观的图形用户界面 (GUI) 以便用户轻松创建和管理提示词。
   - 支持拖放功能以便重新排序提示词。
   - 提供搜索和过滤功能以快速查找特定提示词。
   - 支持关联 prompt 和预览图.

2. **prompt 创建与编辑**:
   - 支持多种提示词格式 (文本、代码片段等)。
   - 提供语法高亮和自动补全功能以提升编辑体验。
   - 支持版本控制以便用户跟踪提示词的修改历史。

3. **prompt单词库**
    - 内置常用提示单词库，用户可以快速选择和组合。
    - 支持用户自定义提示词模板并保存到库中。
    - 提供分类和标签功能以便组织提示词.
